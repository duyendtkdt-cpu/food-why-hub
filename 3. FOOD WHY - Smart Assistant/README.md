# ⚖️ FoodAuditor AI - Trợ lý của Sếp

Ứng dụng hỗ trợ Auditor ngành thực phẩm tra cứu luật, lập checklist và kiểm tra tuân thủ.

## 🚀 Cách chạy ứng dụng

1. Đảm bảo Sếp đã cài đặt Python.
2. Cài đặt các thư viện cần thiết:
   ```bash
   pip install -r requirements.txt
   ```
3. Chạy ứng dụng Streamlit:
   ```bash
   streamlit run food_auditor_app.py
   ```

## ✨ Tính năng chính
- **💬 Hỏi đáp với Em:** Tra cứu nhanh các điều khoản luật và tiêu chuẩn (ISO, HACCP, FSSC...).
- **📋 Lập Checklist nhanh:** Chuyển văn bản thô thành bảng kiểm tra chuyên nghiệp.
- **🔍 Soi Tuân thủ (Audit):** Phân tích hình ảnh/tình huống và trích dẫn luật chính xác.
- **📚 Thư viện Luật:** Lưu trữ và tìm kiếm văn bản pháp luật thực phẩm Việt Nam.
